
## node version
node 14.17.0  
npm 7.16.0

## installation
execute command below in project root directory.

```
$ mkdir build
$ mkdir src/images
$ yarn
```

## start dev server
execute command below in project root directory, and the page will open on your browser automatically.

```
$ yarn start
```

## start production server

```
$ yarn build
$ cd build
$ npx serve
```
